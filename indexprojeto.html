Conceito de HTML:

HTML (HyperText Markup Language) é uma linguagem de marcação utilizada para criar e estruturar o conteúdo de páginas da web. Ela define a organização dos elementos de um site, como títulos, textos, imagens, links, listas e formulários, por meio de tags. O HTML é interpretado pelos navegadores (como Chrome e Firefox), que exibem o conteúdo de forma visual para o usuário. Ele não possui lógica de programação, sendo responsável apenas pela estrutura e semântica da página, enquanto o CSS cuida do visual e o JavaScript da interatividade.

Conceito de Tags HTML:

As tags HTML são elementos usados para marcar e organizar o conteúdo de uma página web. Elas informam ao navegador como o conteúdo deve ser estruturado e exibido, como títulos, parágrafos, listas, links, imagens e outros componentes. As tags geralmente são escritas entre os sinais < (MENOR) e > (MAIOR) e, na maioria dos casos, possuem uma tag de abertura e uma de fechamento.

Tags e Exemplos HTML:

<html>

A tag <html> é a tag raiz de um documento HTML. Ela envolve todo o conteúdo da página e informa ao navegador que aquele arquivo é um documento HTML. Dentro dela ficam as tags <head> e <body>.

Essa tag também pode conter o atributo lang, que define o idioma da página.

Exemplo de uso da tag <html>:

<html>
    <head>
        <title>Título da página</title>
    </head>
    <body>
        <p>Conteúdo da página</p>
    </body>
</html>

<!DOCTYPE html>

A declaração <!DOCTYPE html> informa ao navegador que o documento está utilizando a versão HTML5. Ela deve ser a primeira linha do arquivo HTML.

Essa declaração evita erros de interpretação e garante que o navegador renderize a página corretamente.

Exemplo:

<!DOCTYPE html>
<html lang="pt-BR">

<head>

A tag <head> contém informações sobre o documento, que não aparecem diretamente para o usuário. Ela é usada para configurar metadados da página.

Dentro do <head> ficam tags como: <meta>, <title> e links para CSS.

Exemplo:

<head>
    <meta charset="UTF-8">
    <title>Título da página</title>
</head>

<meta charset>

A tag <meta charset="UTF-8"> define o conjunto de caracteres da página.

Ela garante que acentos e caracteres especiais, como ç, ã, é, sejam exibidos corretamente.

Exemplo:

<meta charset="UTF-8">

<title>

A tag <title> define o título da página, que aparece na aba do navegador.

Esse título também é muito importante para SEO e organização do site.

Exemplo:

<title>Título da página</title>

<body>

A tag <body> contém todo o conteúdo visível da página, como textos, títulos, listas, links e imagens.

Tudo o que o usuário vê no navegador está dentro do <body>.

Exemplo:

<body>
    <h1>Meu primeiro projeto</h1>
    <p>Conteúdo da página</p>
</body>

<p>

A tag <p> é utilizada para criar parágrafos, organizando o texto em blocos para facilitar a leitura.

<strong>

A tag <strong> indica que o texto possui maior importância semântica, normalmente exibido em negrito.

<i>

A tag <i> deixa o texto em itálico, sendo usada para dar ênfase a uma palavra ou frase.

<u>

A tag <u> sublinha o texto.

<mark>

A tag <mark> destaca partes do texto, como se fosse um marca-texto.

<small>

A tag <small> exibe o texto em tamanho menor, geralmente utilizada para observações ou informações complementares.
